# Dados Brutos (Raw Data)  

Este diretório contém os dados brutos utilizados no projeto. Devido ao tamanho (>36GB), os arquivos originais não estão incluídos aqui.  

### **Fontes Oficiais**  
1. **IEEE CIS Fraud Detection Dataset**:  
   - **Link**: https://www.kaggle.com/c/ieee-fraud-detection/data  
   - **Passos para Download**:  
     a. Crie uma conta no Kaggle.  
     b. Acesse a página da competição (link acima).  
     c. Clique em "Download All" para obter os arquivos `train_transaction.csv`, `test_transaction.csv`, etc.  

2. **Credit Card Fraud Detection Dataset**:  
   - **Link**: https://www.kaggle.com/mlg-ulb/creditcardfraud  
   - **Passos para Download**:  
     a. Acesse o link acima.  
     b. Clique em "Download" para obter o arquivo `creditcard.csv`.  

### **Estrutura Esperada**:  
- `ieee_train.csv`  
- `ieee_test.csv`  
- `creditcard.csv`  e, 2019. Disponível em: https://www.kaggle.com/c/ieee-fraud-detection. Acesso em: 26 mar. 2025.
